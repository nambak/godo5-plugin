지정된 파일 또는 디렉토리의 고도몰5 자동패치 호환성을 검사한다.

대상 경로: `$ARGUMENTS` (미지정 시 `module/` 디렉토리 전체)

검사 항목:

1. **Bundle 수정 여부** — Bundle/ 디렉토리 내 파일이 수정되었는지 확인 (git diff 또는 파일 타임스탬프)
2. **상속 패턴** — module/ 내 클래스가 `extends \Bundle\...` 형태로 올바르게 상속하는지 확인
3. **App::load() 사용** — `new \Bundle\...` 직접 생성 코드가 없는지 확인
4. **라이선스 헤더** — 모든 PHP 파일에 NHN godo 라이선스 헤더가 있는지 확인
5. **바인드 쿼리** — SQL 문자열 직접 삽입 패턴이 없는지 확인 (정규식으로 검사)
6. **금지 함수** — `eval()`, `extract()`, `var_dump()`, `print_r()` 사용 여부
7. **금지 접두사** — `es_`, `zz_` 접두사로 시작하는 커스텀 테이블/컬럼 없는지 확인
8. **파일 길이** — 500줄 초과 파일 경고

결과를 표 형식으로 정리하여 보고한다:
- 통과 항목은 ✅
- 위반 항목은 ❌ + 파일 경로 + 구체적 위치
- 개선 권장 항목은 ⚠️

마지막에 전체 호환성 점수(%)를 표시한다.
