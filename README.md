# godomall5-plugins

NHN Godo5(고도몰5) 개발 전문 플러그인 마켓플레이스입니다.

## 설치 방법

### 1. 마켓플레이스 추가

```bash
/plugin marketplace add your-github-id/godo5-plugin
```

### 2. 플러그인 설치

```bash
/plugin install godomall5-developer@godomall5-plugins
```

## 포함된 플러그인

### godomall5-developer

고도몰5 쇼핑몰 개발 전문 플러그인입니다.

**스킬:**
- 고도몰5 아키텍처, 자동패치 호환 커스터마이징, Controller/Component/Widget 확장 패턴, DB 설계 규칙, 프론트엔드 스킨 문법 등 전문 지식 제공

**커맨드:**
- `/godo-init [프로젝트명]` — 새 프로젝트 CLAUDE.md 자동 생성
- `/godo-check [경로]` — 자동패치 호환성 검사
